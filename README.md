
# Blackjack Game.
>Stack React + Typescript + Redux + Saga
>

Implemented normal rules for blackjack, Aces can be 1 or 11.
Game logic handled with redux-sage  


### start play locally
```bash
$> npm start
```


### run tests

```bash
$> npm test 
```

### build for production

```bash
$> npm build
```
